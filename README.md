# Bright Path Tutoring — Simple Launch Guide

This folder contains the actual website files.

## Easiest live launch

1. Go to https://app.netlify.com/drop
2. Drag this entire folder — `WEBSITE-FILES-TO-UPLOAD` — onto the page.
3. Netlify gives you a live URL.

No build step is needed.

## Local preview

Open `index.html` in a browser.

## Files

- `index.html` — homepage
- `contact.html` — consultation/contact page
- `styles.css` — design
- `script.js` — contact form text-message behavior
